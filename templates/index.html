How to Play

Use Python code to move your chess pieces (white). The computer will automatically play black.

To move a piece, use the move_piece(start, end) function:

  • start: The starting position (e.g., 'e2')
  • end: The target position (e.g., 'e4')
Example Moves:
# Move pawn from e2 to e4
move_piece('e2', 'e4')

# Move knight from g1 to f3
move_piece('g1', 'f3')
